Sorting Algorithms

Bubble Sort

A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Visualize Bubble Sort

Selection Sort

Sorts an array by repeatedly finding the minimum element from the unsorted portion and placing it at the beginning.

Visualize Selection Sort

Insertion Sort

Builds the final sorted array one item at a time by comparing each element with the ones before it.

Visualize Insertion Sort

Merge Sort

A divide-and-conquer algorithm that recursively breaks down a problem into smaller, more manageable subproblems until they become simple enough to solve directly.

Visualize Merge Sort

Quick Sort

An efficient, in-place sorting algorithm that uses a divide-and-conquer strategy to sort elements around a pivot.

Visualize Quick Sort
