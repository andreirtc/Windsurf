How Bubble Sort Works

Bubble Sort is a simple comparison-based sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Algorithm Steps:

  1. Start with the first element and compare it with the next element
  2. If the first element is greater than the second element, swap them
  3. Move to the next pair of elements and repeat step 2
  4. Continue this process until the end of the array
  5. Repeat steps 1-4 until no more swaps are needed

Pseudocode:

method bubbleSort(array A, integer N)
    for each R from N-1 down to 1        // repeat for N-1 iterations
        for each i in [0..R-1]           // the 'unsorted region'
            if A[i] > A[i+1]             // these two are not in order
                swap(A[i], A[i+1])       // swap them in O(1)
Total: O(N²) = (N-1)+(N-2)+...+1 iterations
